# Premium To-Do App

A modern, aesthetically pleasing To-Do application built with vanilla HTML, CSS, and JavaScript.

## Features
- **Premium UI**: Glassmorphism design with animated background.
- **Theme Customization**: Toggle between Light and Dark modes.
- **Interactive**: Drag & Drop sorting and Double-click to Edit.
- **Categories**: Organize tasks by Personal, Work, Shopping, etc.
- **Progress Tracking**: Real-time progress bar and celebration effects.
- **Task Management**: Add, complete, and delete tasks.
- **Persistence**: Tasks are saved to LocalStorage.
- **Filtering**: View All, Active, or Completed tasks.
